<h1>Installation Steps</h1>

The following libraries need to be globalled installed for this repo to work

<ul>
  <li>Typescript</li>
  <li>Nodemon</li>
</ul>

---

<h3>1. Packages to Install</h3>

```
$ yarn install
```

<h3>2. Populate ENV</h3>

```
$ yarn setup_env
```

Please provide values in the newly created .env file

<h3>3. Run App</h3>

```
$ yarn build
$ yarn start
```
